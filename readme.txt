Show Nxt assets information on yours Wordpress sites.

== Description ==

The plugin show information about owner account like official client. It can generate unsigned transaction for send Nxt action or send already signed transaction into the Nxt network. 

Additionally you can download our [NXTBridge AE](https://wordpress.org/plugins/nxtbridge/) plugin to show Assets information on your site, like this: https://nxter.org/assethub/


== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be:
  './wp-content/plugins/nxtbridge-ledger/*'

  Activate plugin at "Plugins" administration page.

== About this project ==

NxtBridge-Ledger is a WordPress plugin series which can turn your Wordpress site into a Nxt Wallet. You can create unsigned transactions and send locally signed transcations to the Nxt network without any additionally software.
